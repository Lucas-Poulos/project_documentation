# Electrical Projects Documentation

This repository serves as a centralized storage for documentation related to electrical projects, including PDFs, schematics, datasheets, and other technical documents.

## Repository Structure

- **datasheets/** - Component datasheets and specifications
- **schematics/** - Circuit diagrams and electrical schematics
- **manuals/** - User manuals and technical guides
- **projects/** - Individual project documentation organized by project name
- **reference/** - General reference materials and standards

Documentation for power converter's project.

## Usage

To add new documentation:
1. Place files in the appropriate folder based on the structure above
2. For new projects, create a subfolder in `projects/` with the project name
3. Update this README with a brief description in the Projects section

## File Types

This repository supports various documentation formats:
- PDF documents
- Images (PNG, JPG, SVG)
- CAD files
- Spreadsheets
- Text documents
- Circuit design files
